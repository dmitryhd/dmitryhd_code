SMB Gate

Q. Что такое SMB Gate?

A. SMB Gate - односторонний шлюз между SMB и HTTP. Он позволяет использовать ресурсы сети на основе протокола SMB (обычная виндовая сеть) если у вас есть доступ туда только по HTTP. Скрипт достает по SMB запрашиваемый вами файл и высылает вам его по HTTP.

Q. Как использовать SMB Gate?

A. Формат строки запроса следующий: http://lorien/smb/< host >/< share >/< file > где

  • http://lorien/smb/ - местонахождение шлюза.
  • <host> - имя машины в сетевом окружении
  • <share> - имя расшаренного ресурса
  • <file> - имя нужного файла

Также можно посмотреть список файлов в каталоге. Например, зайдите на http://lorien/smb/lorien/video. Для более полноценной навигации по сети используйте Fast SMB Search (для быстрого попадания в нужный каталог служит кнопка View this directory in Fast SMB Search внизу страницы).
Если вы используете Microsoft Internet Explorer, не забудьте выключить режим Always send URL as UTF-8.
